## 架构要点（防踩坑）

- **IPC 唯一通道**：渲染进程禁止直接 `fetch`，所有 Bilibili 请求走 IPC → 主进程
- **preload 双文件同步**：`preload.ts` 和 `preload.cjs` 必须同步修改
- **design-tokens 唯一颜色源**：组件样式用 CSS 变量，不硬编码颜色
- **electron-store 类型不共享**：main.ts schema 与渲染进程类型各自维护，修改时需同步
- **audio URL 过期**：~10 分钟有效期，播放器每 60s 自动刷新
- **noUncheckedIndexedAccess**：数组索引返回 `T | undefined`，已知值用 `!` 断言
